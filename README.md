# DonaldsonPlumbingV22_Martin
Initial setup using Claude AI
This project is a modern, fully responsive marketing website for 
Donaldson's Plumbing & Heating, built using HTML, CSS, and JavaScript.

The site includes four main pages:
- Home
- Services
- About
- Contact



# Important information
- Built with mobile in mind 
- CSS Grid for service cards and content sections
- Flexbox for navigation and hero content
- Clamp() typography for fluid scaling



# Design Information 
You can modify:
- Colors → in :root
- Typography → Google Fonts links
- Hero images → page‑specific classes
- Buttons → .btn, .btn-primary, .btn-secondary
- Cards → .service-card, .value-card, .metric-card
The design system is built to scale.
